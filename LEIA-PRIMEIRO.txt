🆘 SE NÃO FUNCIONAR
────────────────────────────────────────────────────────────────────────────

  Erro ao executar STARTUP.bat?
    → Executar como Administrador
    → Verificar se Docker está rodando
    → Consultar SETUP-LOCAL.md

  Porta 4173 já em uso?
    → Editar .env: trocar PORT=4173 para PORT=4174

  Não consegue acessar localhost:4173?
    → Esperar 2-3 minutos
    → Verificar console para erros
    → Consultar SETUP-LOCAL.md (Troubleshooting)

  Login não funciona?
    → Usuário: "Administrador" (exatamente assim)
    → PIN: "0000" (4 zeros)
    → Ou PIN: "1111" para técnicos
