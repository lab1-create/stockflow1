### Método 1: GitHub (Recomendado para Equipe)
- [ ] Repositório GitHub criado
- [ ] Código enviado para GitHub
- [ ] `.env` não está no repositório (git ignora)

- [ ] Enviar instrução: "Clone e execute STARTUP.bat"

### Método 2: ZIP (Para Entrega Única)
- [ ] Remover `node_modules`
- [ ] Remover `.env` local
- [ ] Compactar pasta
- [ ] Verificar `.env.example` está no ZIP
- [ ] Enviar arquivo `stockflow-main.zip`
- [ ] Enviar instrução: "Descompacte e execute STARTUP.bat"

- [ ] Pasta sincronizada na nuvem
- [ ] `.env` não está sincronizado (local only)
- [ ] Gerar link compartilhado
- [ ] Enviar instrução: "Baixe/sincronize e execute STARTUP.bat"
